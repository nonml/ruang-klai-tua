# Thai Safe Wording (Defamation Risk Reduction)

## Tone guideline
- Neutral
- Evidence-first
- No accusation

## Preferred patterns
- "ข้อมูลจากหลักฐาน ณ วันที่..."
- "มีผู้รายงานว่า..." (as unverified statement framing)
- "โปรดระบุข้อเท็จจริงที่พบเห็น"
- "หลีกเลี่ยงการระบุชื่อบุคคล/ข้อมูลส่วนตัว"

## Avoid patterns
- Direct accusation terms (e.g., criminal-claim wording)
- Language that implies guilt before verification
- Personal targeting or naming when unnecessary

## UI reminders
- Show short warning near text input.
- Keep submit-page disclaimer visible.
- Mark report state clearly: pending/held/verified/hidden.
